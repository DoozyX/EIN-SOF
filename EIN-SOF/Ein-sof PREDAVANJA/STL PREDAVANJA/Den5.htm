4.5 Секвенцијални контејнери

 

Standard Template Library (STL) содржи три секвенцијални контејнери: vector, list и deque. vector и deque се базираат на полиња. list е имплементирана како поврзана листа.

Еден од најпопуларните контејнери во STL е vector. vector може да ја менува големината динамички. Вредноста на еден vector може да биде доделена на друг vector. Не се врши автоматска проверка на индексот, но тоа е можно преку функцијата членка at.

Вметнување на елемент на крај на vector е ефикасно. Вметнување или бришење на елемент на средина е скапа операција бидејќи бара поместување на елементите на vector-от, бидејќи тие се последователно сместени во меморијата.

Освен операциите што се заеднички за сите контејнери, секој контејнер има свои додатни можности. Тие не се подеднакво ефикасни кај сите контејнери. Затоа програмерот мора да одбере кој контејнер е најсоодветен за неговата апликација.

Апликации кои бараат често вметнување и бришење на двата краја на контејнерот обично користат deque бидејќи таа врши поефикасно додавање на почетокот. Апликации кои бараат често вметнување и бришење во средината на контејнерот обично користат list.

Секвенцијалните контејнери содржат неколку заеднички операции: front враќа референца на првиот елемент, back враќа референца на последниот елемент, push_pack додава нов елемент на крајот и pop_back го отстранува последниот елемент.

4.5.1 Секвенцијален контејнер vector

Класата vector обезбедува податочна структура која се состои од последователни елементи на кои може да им се пристапува преку операторот []. Таа најчесто се користи кога податоците треба да се организирани во сортирана низа на елементи на кои треба да им се обезбеди случаен пристап.

vector поддржува random-access итератори, што значи дека ги вклучува сите итераторски операции. Сите STL алгоритми можат да се извршуваат врз vector. Итераторите за vector се имплементирани како покажувачи на елементите на vector.

Почетно, vector има големина 0. Кога се додаваат нови елементи во него, vector алоцира поголем континуиран мемориски блок, ги копира елементите во него и го деалоцира стариот мемориски блок.

Следниот пример илустрира употреба на vector:

//

// Demonstriranje na STL class template vector.

 

#include <iostream>

 

using std::cout;

using std::cin;

using std::endl;

 

#include <vector>  // vector class-template definicija

 

// prototip za function template printVector

template < class T >

void printVector( const std::vector< T > &integers2 );

 

int main()

{

   const int SIZE = 6;  

   int array[ SIZE ] = { 1, 2, 3, 4, 5, 6 };

 

   std::vector< int > integers;

 

   cout << "Pocetnata golemina na vector-ot integers e: "

        << integers.size()

        << "\nPocetniot kapacitet na vector-ot integers e: "

        << integers.capacity();

 

   // Dodavanje na elementi

   integers.push_back( 2 ); 

   integers.push_back( 3 ); 

   integers.push_back( 4 );

 

 

   cout << "\nGoleminata na vector-ot integers e: " << integers.size()

        << "\nKapacitetot na vector-ot integers: "

        << integers.capacity();

 

   cout << "\n\nPrikaz na pole preku pokazuvaci: ";

 

   for ( int *ptr = array; ptr != array + SIZE; ++ptr )

      cout << *ptr << ' ';

 

   cout << "\nPrikaz na vector preku iterator: ";

   printVector( integers );

 

   cout << "\nInvertirana sodrzina na vector: ";

 

   std::vector< int >::reverse_iterator reverseIterator;

 

   for ( reverseIterator = integers.rbegin();

         reverseIterator!= integers.rend();

         ++reverseIterator )

      cout << *reverseIterator << ' ';

 

   cout << endl;

 

   return 0;

 

} // kraj na main

 

 

// function template za prikazuvanje na elementi na vector

template < class T >

void printVector( const std::vector< T > &integers2 )

{

   std::vector< T >::const_iterator constIterator;

 

   for ( constIterator = integers2.begin();

         constIterator != integers2.end();

         constIterator++ )

      cout << *constIterator << ' ';

 

} // kraj na printVector

Pocetniot kapacitet na vector-ot integers e: 0

Goleminata na vector-ot integers e: 3

Kapacitetot na vector-ot integers: 4

 

Prikaz na pole preku pokazuvaci: 1 2 3 4 5 6

Prikaz na vector preku iterator: 2 3 4

Invertirana sodrzina na vector: 4 3 2

За да се користи vector треба да се вклучи загавието <vector>. Во програмата седефинира инстанца integers на класата vector во која се сместуваат int вредности. Кога се инстанцира објектот се креира празен vector со големина 0 и капацитет 0, чии вредности се прикажуваат со функциите size и capacity.

Со push_back се додаваат нови елементи на крајот на vector. Ако се додава елемент на полн vector, тогаш vector-от ја зголемува својата големина. Кај некои STL имплементации големината се зголемува двојно. (Во тој случај, капацитетот во примерот треба да стане 8 со додавање на уште два елементи). Но, ова може да значи непотребно алоцирање на простор. Тоа може да се избегне со употреба на resize.

Функцијата printVector прикажува vector преку итератор. Таа како аргумент прима const референца на vector. Потоа се дефинира const_iterator (програмата може само да ги прикажува елементите на vector, но не и да ги менува) constIterator кој минува низ vector-от и ја прикажува неговата содржина.

integers2.begin() враќа const_iterator на првиот елемент на vector. Крајот на vector го определува integers2.end().

Само random-access инератори поддржуваат <. Затоа е подобро да се користи != и end() за проверка на крај на контејнер.

За поминување на vector во обратен редослед се дефинира reverse_iterator. Слично како кај printVector, се користат  rbegin и rend за определување на почетокот и на крајот на vector-от во обратен редослед.

Следниот пример ги демонстрира функциите за манипулација со елементите на vector.

//

// Testiranje na funkcii za manipulacija so elementi na class template vector

 

 

#include <iostream>

 

using std::cout;

using std::endl;

 

#include <vector>    

#include <algorithm>

 

int main()

{

   const int SIZE = 6;  

   int array[ SIZE ] = { 1, 2, 3, 4, 5, 6 };

 

   std::vector< int > integers( array, array + SIZE );

   std::ostream_iterator< int > output( cout, " " );

   // iterator koj prikazuva int na cout so separator " "

 

   cout << "Vector integers sodrzi: ";

   std::copy( integers.begin(), integers.end(), output );

 

   cout << "\nPrv element na integers: " << integers.front()

        << "\nPosleden element na integers: " << integers.back();

 

   integers[ 0 ] = 7;      // Postavi vrednost 7 na prv el.

   integers.at( 2 ) = 10;  // Postavi vrednost 10 na el. na pozicija 2

 

// pristap do out-of-range element

   try {

      integers.at( 100 ) = 777;  

   } // end try

   // ulovi out_of_range exception

   catch ( std::out_of_range outOfRange ) {

      cout << "\n\nException: " << outOfRange.what();

 

   } // end catch

   // Vmetni 22 na pozicija 2

   integers.insert( integers.begin() + 1, 22 ); 

 

   cout << "\n\nVector integers sodrzi: ";

   std::copy( integers.begin(), integers.end(), output );

 

   // izbrisi go prviot element

   integers.erase( integers.begin() );

   cout << "\n\nVector integers sodrzi: ";

   std::copy( integers.begin(), integers.end(), output );

 

   // izbrisi gi site elementi

   integers.erase( integers.begin(), integers.end() );

   cout << "\nPo brisenje na site elementi integers"

        << ( integers.empty() ? " e" : " ne e" ) << " prazno";

 

   // dodaj elementi od poleto

   integers.insert( integers.begin(), array, array + SIZE );

   cout << "\n\nVector integers sodrzi: ";

   std::copy( integers.begin(), integers.end(), output );

 

   //

   integers.clear(); 

   cout << "\nPo praznenje, integers"

        << ( integers.empty() ? " e" : " ne e" ) << " prazno";

 

 

   cout << endl;

 

   return 0;

 

} // kraj na main

 

Излезот на програмата е:

 

Vector integers sodrzi: 1 2 3 4 5 6

Prv element na integers: 1

Posleden element na integers: 6

 

Exception: invalid vector<T> subscript

 

Vector integers sodrzi: 7 22 2 10 4 5 6

 

Vector integers sodrzi: 22 2 10 4 5 6

Po brisenje na site elementi integers e prazno

 

Vector integers sodrzi: 1 2 3 4 5 6

Po praznenje, integers e prazno

 

Линијата:

 

  std::vector< int > integers( array, array + SIZE );

 

користи конструктор на vector со два аргументи (покажувач на почетокот на array и покажувач на крајот на array) за иницијализација на објектот.

За прикажување на integers на cout се користи STL алгоритам copy.

 

std::copy( integers.begin(), integers.end(), output );

 

Аргументи на copy се три итератори, почетокот на integers integers.begin(), еден елемент после последниот елемент на integers integers.end() и излезен итератор output.

front и back се користат за да се добие првиот и последниот елемент на integers.

Има два начини на пристап до елементите на vector, преку subscript операторот [] и преку функцијата at (at врши и проверка дали индексот е во границите на полето. Ако не е, фрла исклучок):

 

integers[ 0 ] = 7;     

   integers.at( 2 ) = 10;

 

insert врши вметнување на елемент пред локацијата на која покажува итераторот. Постои верзија на insert која вметнува повеќе копии на една вредност и уште една верзија со која се вметнуваат низа вредности од друг контејнер:

 

   // dodaj elementi od poleto

   integers.insert( integers.begin(), array, array + SIZE );

 

erase е функција која ја имаат сите првокласни контејнери. Во програмата се користи  за бришење на еден елемент:

 

   // izbrisi go prviot element

   integers.erase( integers.begin() );

 

и за бришење на повеќе елементи во даден интервал:

 

   // izbrisi gi site elementi

   integers.erase( integers.begin(), integers.end() );

 

empty се користи за проверка дали контејнерот е празен.

 

Посотојат исто така други функции кои се заеднички за сите контејнери, а не се споменати досега. Дел од нив ќе бидат објаснети во продолжение.

 

4.5.2 Секвенцијален контејнер list

 

Контејнерот list обезбедува ефикасна импламентација на вметнување и бришење на било која локација на контејнерот. Класата е имплементирана како двојно поврзана листа. Затоа таа поддржува употреба на двонасочни (bidirectional) итератори.

Покрај функциите заеднички за сите STL контејнери:

default конструктор

copy конструктор

destructor

empty – враќа true ако нема елементи воконтејнерот

max_size – враќа максимален број на елементи во контејнерот

size

operator=

operator<

operator<=

operator>

operator>=

operator==

operator!=

swap

 

и оние заеднички за секвенцијалните контејнери: front, back, push_pack и pop_back,

list поддржува уште осум други функции членки: splice, push_front, pop_front, remove, unique, merge, reverse и sort.

За да се користи list треба да се вклучи загавието < list >. Следната програма илустрира употреба на контејнерот list.

 

 

// Test programa za STL list class template.

#include <iostream>

 

using std::cout;

using std::endl;

 

#include <list>       // list class-template definicija

#include <algorithm>  // se vklucuva bidejki se koristi algoritmot copy

 

// prototip za function template printList

template < class T >

void printList( const std::list< T > &listRef );

 

int main()

{

   const int SIZE = 4;

   int array[ SIZE ] = { 2, 6, 4, 8 };

 

   std::list< int > values;

   std::list< int > otherValues;

 

   // vmetnuvanje elementi vo listata values

   values.push_front( 1 );

   values.push_front( 2 );

   values.push_back( 4 );

   values.push_back( 3 );

  

   cout << "values sodrzi: ";

   printList( values );

 

   values.sort();  // sortiraj

 

   cout << "\nvalues po sortiranje: ";

   printList( values );

 

   // vmetni gi elementite od poleto array vo otherValues

   otherValues.insert( otherValues.begin(),

      array, array + SIZE );

 

   cout << "\notherValues po dodavanje elementi: ";

   printList( otherValues );

 

   // ostrani elementi od otherValues i dodaj gi na kraj na values

   values.splice( values.end(), otherValues );

  

   cout << "\nvalues po splice: ";

   printList( values );

cout << "\notherValues po splice: ";

   printList( otherValues );

 

   values.sort();  // sortiraj go values

 

   cout << "\nvalues po sort: ";

   printList( values );

 

   // vmetni gi elementite od poleto array vo otherValues

   otherValues.insert( otherValues.begin(),

      array, array + SIZE );

   otherValues.sort();

  

   cout << "\notherValues po dodavanje elementi: ";

   printList( otherValues );

  

   // ostrani elementi od otherValues i dodaj gi vo values sortirani

   values.merge( otherValues );

  

   cout << "\nPo merge:\n   values sodrzi: ";

   printList( values );

   cout << "\n   otherValues sodrzi: ";

   printList( otherValues );

 

   values.pop_front();  // otstrani go elementot na pocetok

   values.pop_back();   // otstrani go elementot na kraj

  

   cout << "\nPo pop_front i pop_back:"

        << "\n   values sodrzi: ";

   printList( values );

  

   values.unique();  // ostrani dupli elementi

  

   cout << "\nPo unique, values sodrzi: ";

   printList( values );

 

   // razmeni gi elementite na values i otherValues

   values.swap( otherValues );

  

   cout << "\nPo swap:\n   values sodrzi: ";

   printList( values );

   cout << "\n   otherValues sodrzi: ";

   printList( otherValues );

 

   // zameni ja sodrzinata na values so elementi od otherValues

   values.assign( otherValues.begin(), otherValues.end() );

  

   cout << "\nPo assign, values sodrzi: ";

   printList( values );

 

    // ostrani elementi od otherValues i dodaj gi vo values sortirani

 

   values.merge( otherValues );

  

   cout << "\nPo merge, values sodrzi: ";

   printList( values );

  

   values.remove( 4 );  // otstrani ki site 4-ki

  

   cout << "\nPo remove( 4 ), values sodrzi: ";

   printList( values );

 

   cout << endl;

 

   return 0;

 

} // kraj na main

 

// printList function template definicija; koristi

// ostream_iterator i copy algoritam za pecatenje na elementi

template < class T >

void printList( const std::list< T > &listRef )

{

   if ( listRef.empty() )

      cout << "Listаta e prazna";

 

   else {

      std::ostream_iterator< T > output( cout, " " );

      std::copy( listRef.begin(), listRef.end(), output );

 

   }

 

} // kraj na printList

Верзијата на sort која се користи тука врши сортирање на елементите во растечки редослед. Оваа функција е различна од STL алгоритмот sort.

 

   // ostrani elementi od otherValues i dodaj gi na kraj na values

   values.splice( values.end(), otherValues );

 

 

splice отстранува елементи од otherValues и ги додава во values на позицијата определена со итераторот values.end(), кој е прв аргумент на функцијата. Има уште две верзии на splice.

   // ostrani elementi od otherValues i dodaj gi vo values sortirani

   values.merge( otherValues );

 

Со merge се отстрануваат елементите од otherValues и се додаваат сортирани во values. Пред да се изврши оваа функција двете листи мораат да бидат сортирани.

   values.unique();  // ostrani dupli elementi

 

Со unique се отстрануваат дуплите елементи од листата. Таа морапретходно да биде сортирана.

 

   // zameni ja sodrzinata na values so elementi od otherValues

   values.assign( otherValues.begin(), otherValues.end() );

 

assign ја заменува содржината на values со содржината на otherValues во интервалот определен со двата итератори. Во другата верзија на assign се заменува содржината на контејнерот со една вредност. Првиот аргумент е бројот на копии што се доделуваат, а вториот е вредноста што се доделува.

 

4.5.3 Секвенцијален контејнер deque

 

Класата deque (dоuble-ended-queue, deck) е слична на vector, обезбедува индексиран пристап до елементите. Оптимизирана е за ефикасно додавање и бришење на елементи на почетокот и на крајот. Поддржува random-access итератори, па затоа може да се користи со сите STL алгоритми. Најчесто, deque се употребува за импламентација на FIFO редица.

Додатен простор за сместување на елементи може да се алоцира на двата краја на редицата, во мемориски блокови. Најчесто пристап до меморијата се реализира преку покажувачи до мемориските блокови. Меморијата вокоја се сместува deque не е континуирана. Најчесто, таа не се деалоцира се додека објектот не се уништи за да се постигне поголема ефикасност во оперирањето со елемените, но значи помала ефикасност во искористувањето на меморијата.

Покрај основните операции кои ги поддржува и vector, deque содржи и функции push_front за додавање на елемент на почетокот и pop_front за бришење на елементот на почетокот.

За да се користи deque треба да се вклучи загавието < deque >. Следната програма илустрира употреба на контејнерот deque.

 

//

// Test programa za STL klasa deque

#include <iostream>

 

using std::cout;

using std::endl;

 

#include <deque>      // deque class-template definicija

#include <algorithm>  // se vklucuva bidejki se koristi algoritmot copy

 

int main()

{

   std::deque< double > values;

   std::ostream_iterator< double > output( cout, " " );

 

   // vmetni elementi vo values

   values.push_front( 2.2 );

   values.push_front( 3.5 );

   values.push_back( 1.1 );

 

   cout << "values sodrzi: ";

 

   // upotrebi subscript operator za prezemanje na vrednostite na elementite

   for ( int i = 0; i < values.size(); ++i )

      cout << values[ i ] << ' ';

 

   values.pop_front();  // otstrani go prviot element

 

   cout << "\nPo pop_front, values sodrzi: ";

   std::copy( values.begin(), values.end(), output );

 

   // upotrebi subscript operator za promena na vrednosta

   // na elementot lokacija 1

   values[ 1 ] = 5.4;

 

   cout << "\nPo values[ 1 ] = 5.4, values sodrzi: ";

   std::copy( values.begin(), values.end(), output );

  

   cout << endl;

 

   return 0;

 

} // kraj na main

 

Излезот од програмата е:

 

values sodrzi: 3.5 2.2 1.1

Po pop_front, values sodrzi: 2.2 1.1

Po values[ 1 ] = 5.4, values sodrzi: 2.2 5.4

Проект 1:

 

Во едно множество се сместуваат објекти кои се геометриски фигури. Креирај класи Pravoagolnik, Kvadrat и Krug. Креирај class template Mnozestvo<T> во со функции членки за додавање, прикажување на елементи на множеството и определување на најголем елемент во множеството (елемент со најголема површина). Како податочна структура за сместување на елементите на множеството може да се користи поле. Секој елемент може да се појави само еднаш. Креирај три множества: pravoagolnici, kvadrati и krugovi од Pravoagolnik, Kvadrat и Krug, соодветно. Креирај function template за печатење на множество. Програмата треба да имплементира мени со следните опции:

 

1.      Додај правоаголник

2.      Додај квадрат

3.      Додај круги

4.      Прикажи го множеството правоаголници

5.      Прикажи го множеството квадрати

6.      Прикажи го множеството кругови

7.      Прикажи најголеми елементи

8.      Прикажи број на елементи во множествата

9.      Крај
